# Sudoku

Sudoku is a programming for solving sudoku, written in Coffeescript.

## Installation

```
$ git clone https://github.com/andersjanmyr/sudoku.git
$ npm install
```

## Test

```
# Run all tests
$ npm test

# Watch files and run tests when files change
$ npm run watch
```

## Running

```
$ npm run easy
$ npm run medium
$ npm run hard
$ npm run samurai
```
